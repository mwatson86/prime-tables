# Prime Tables

### Installation

```sh
$ npm install
```

### Development build

```sh
$ npm run dev
```

Navigate to http://localhost:3000 in a browser of your choosing

### Production build

```sh
$ npm run build
$ npm start
```

Navigate to http://localhost:3000 in a browser of your choosing
